GPSLogger for Android

A battery efficient GPS logging application for Android

GPSLogger uses the GPS capabilities of your Android phone to log coordinates to a GPS or KML file at regular intervals. This can be particularly useful if you want to geotag your photos after a day out or share your travel route with someone. The purpose of this application is to be battery efficient to save you battery power when abroad and last as long as possible.

Quick Tour

Menu and icons

  • Use the views dropdown to navigate between the different presentations.
  • While you are logging, the green indicator will stay on.
  • The spinner only appears when the GPS is trying to get a fix.

Simple view

  • Shows your last recorded GPS point, updates when new point available
  • Tap any field to highlight the text
  • Big start/stop button. You can't miss it
  • Distance travelled is the total distance between all points.

Menus

  • Many menu items
  • General settings - Startup, imperial, debug file and version info
  • Logging settings - File formats, folder, new file creation, frequency, accuracy
  • Upload settings - Settings for various destinations such as Google Docs, FTP, Dropbox

Detailed view

  • Detailed view of last recorded GPS point
  • Smaller start/stop button

Big View

  • Big text
  • Meant for landscape mode
  • Awkward in portrait mode
